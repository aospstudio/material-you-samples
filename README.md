# material-you-sample
 Material You interface with dynamic theme support for your Android projects

## Requirements
```bash
Android Studio Chipmunk 2021.2.1 or higher,
Gradle version: 7.2.0,
SDK API 32,
Platform Tools: 33.0.0 rc04,
AndroidX libraries,
Kotlin
```
